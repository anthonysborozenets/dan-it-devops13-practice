useradd виколристовував щоб створити користувачів
groupadd виколристовував щоб створити групи
usermod -aG використовував для додання в групи
mkdir + chown використовував для створення домашніх каталогів та змін прав для користувачів
rsync використовував для користувача backupdev щоб клонувати директорію /dev1

charttr +a використав для того щоб забрати права на видалення і щоб можна було тільки додати в файл. Права на файл дав 666
